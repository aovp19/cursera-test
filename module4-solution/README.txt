Module 4 Coding Assignment

Curso: HTML, CSS, and JavaScript for Web Developers

    Descripción

    Este proyecto practica JavaScript recorriendo un array de nombres y mostrando saludos en la consola:

        J o j → Goodbye JSomeName
        Otros → Hello SomeName

    Se utilizan librerías externas (speakHello.js y speakGoodbye.js) que deben corregirse siguiendo los pasos en script.js.

    Archivos Principales
        module4-solution/
        ├── index.html
        ├── script.js
        ├── speakHello.js
        ├── speakGoodbye.js
        └── README.md

    Uso

    Abrir index.html en un navegador.
    Abrir la consola del navegador (F12) para ver los saludos.
    Seguir los pasos en script.js para completar la funcionalidad.
