# Module 3 Solution

This repository contains my solution for **Module 3 Assignment** of the course:

**HTML, CSS, and Javascript for Web Developers**

Offered by **Coursera**.

## Description

The project demonstrates the use of **Bootstrap** to create a responsive webpage with a navigation menu that collapses on smaller screens.

The page includes:

* A responsive navigation bar
* A collapsible mobile menu
* A menu section layout
* Basic Bootstrap styling

## Technologies Used

* HTML5
* CSS3
* Bootstrap

# Module 4 Solution

This repository contains my solution for Module 4 Assignment of the course:

HTML, CSS, and Javascript for Web Developers

Offered by Coursera.

# Description

The project demonstrates the use of JavaScript to loop over an array of names and print personalized greetings in the browser console.

The logic includes:

Printing Goodbye JSomeName if the name starts with J or j

Printing Hello SomeName for all other names

Fixing and using external libraries (speakHello.js and speakGoodbye.js)

# Technologies Used

HTML5

CSS3

JavaScript
